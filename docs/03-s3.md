# S3

`S3Storage` принимает конфигурацию, совместимую с S3:

- `bucket` — имя бакета.
- `key` — access key.
- `secret` — secret key.
- `endpoint` — URL S3 endpoint (по умолчанию `https://storage.yandexcloud.net`).
- `region` — регион (по умолчанию `ru-central1`).
- `prefix` — общий префикс для ключей (опционально).
- `use_path_style_endpoint` — использовать path-style URL (по умолчанию `true`).
- `baseUrl` — опциональный публичный URL (например CDN).

Пример с явной настройкой:

```php
use PhpSoftBox\Storage\Drivers\S3\S3Storage;

$disk = S3Storage::fromConfig([
    'bucket' => 'my-bucket',
    'key' => $_ENV['S3_KEY'],
    'secret' => $_ENV['S3_SECRET'],
    'endpoint' => 'https://storage.yandexcloud.net',
    'region' => 'ru-central1',
    'use_path_style_endpoint' => true,
]);
```

`url()` вернёт ссылку, сформированную из `baseUrl` (если задан) или из `endpoint`/`bucket`.
