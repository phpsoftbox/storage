# DI

## Пример с PHP-DI

```php
use DI\ContainerBuilder;
use PhpSoftBox\Storage\Storage;

$builder = new ContainerBuilder();
$builder->addDefinitions([
    Storage::class => function (): Storage {
        return new Storage([
            'default' => 'local',
            'disks' => [
                'local' => [
                    'driver' => 'local',
                    'rootPath' => __DIR__ . '/storage',
                    'baseUrl' => 'https://cdn.local',
                ],
            ],
        ]);
    },
]);
```
